این‌ها را داخل build.gradle.kts موجود (app) اضافه/ادغام کنید — جایگزین کل فایل نکنید.

۱) داخل بلاک android { ... }، بعد از defaultConfig یا هرجای مناسب، این را اضافه کنید:

    ndkVersion = "26.1.10909125"

    externalNativeBuild {
        ndkBuild {
            path = file("src/main/jni/Android.mk")
        }
    }

    defaultConfig {
        ndk {
            abiFilters += listOf("arm64-v8a", "armeabi-v7a", "x86", "x86_64")
        }
    }

۲) داخل بلاک dependencies { ... } این خط را اضافه کنید (aar محلی که CI دانلودش می‌کند):

    implementation(files("libs/libv2ray.aar"))

نکته: ndkVersion بالا («26.1.10909125») نسخه‌ای است که روی ubuntu-latest GitHub Actions معمولاً از قبل نصب است؛
اگر CI گفت این نسخه پیدا نشد، در پیام خطا لیست نسخه‌های موجود را نشان می‌دهد — همان یکی را جایگزین کنید.
این تنها بخشی است که واقعاً نمی‌توانم بدون دیدن اجرای واقعی CI، صد‌درصد تضمین کنم.
